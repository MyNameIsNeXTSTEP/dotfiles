# Dotfiles

This repository contains my configurations files on:
- tmux: [.tmux.conf](https://github.com/MyNameIsNeXTSTEP/dotfiles/blob/master/.tmux.conf)
- neovim: [.config/nvim](https://github.com/MyNameIsNeXTSTEP/dotfiles/tree/master/.config/nvim)
- brew: [.brewfile](https://github.com/MyNameIsNeXTSTEP/dotfiles/blob/master/.brewfile)
- zsh/bash [.bashrc](https://github.com/MyNameIsNeXTSTEP/dotfiles/blob/master/.bashrc)
- macos (*to defined later*)

To update dtfiles locally in the repository i use [import.sh](https://github.com/MyNameIsNeXTSTEP/dotfiles/blob/master/import.sh) script.

> Notice:
> The import mechanism considers that dotfiles are taken from the same places as declared in the script, i.e. from `~/` directory.
